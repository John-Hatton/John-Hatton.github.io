Detailed Description

The Half Adder class is an abstraction for a half adder circuit. The internal design uses one XOR gate, and one AND gate.

std::string HalfAdder::answer ( )

The answer function is to be interpreted as the result of the inputs given.

Returns
Returns a string representing the output of the inputs given. For example with the inputs 0 and 1, the generated result would be "01".
