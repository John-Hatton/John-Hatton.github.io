Detailed Description

The Two Bit Adder class represents two Full Adders, chained to get to add two two-bit binary numbers.

std::vector< bool > TwoBitAdder::answer ( )

The answer function is to be interpreted as the result of the inputs given.

Returns
Returns a string representing the output of the inputs given. For example with the inputs 01 and 11, the generated result would be "0 0100". See the test cases in the test folder for more examples of this.
