Detailed Description

The Full Adder class is an abstraction for a full adder. The internal design uses two XOR gates, two AND gates, and one OR gate to accomplish this. I might experiment later, and create a NAND gate, to try to make some of these same circuits with just NAND logic, to see how effective that might be.
